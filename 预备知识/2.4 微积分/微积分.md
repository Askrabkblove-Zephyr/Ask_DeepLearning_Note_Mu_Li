**数值求解**：

```python
import sympy as sp
x1, x2 = sp.symbols('x1 x2')
f = 3*x1**2+5*sp.exp(x2)
grad_f = [sp.diff(f, var) for var in (x1, x2)]
print("函数 f(x1, x2) =", f)
print("梯度 ∇f =", grad_f)
print("即：(", grad_f[0], ",", grad_f[1], ")")
```
